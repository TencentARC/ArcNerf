# img_metric
Metric for evaluating image quality
## PSNR
For general image
### PSNRCoarse
For coarse image
### PSNRFine
For fine image
## MaskPSNR
For general image, with mask
### MaskPSNRCoarse
For coarse image, with mask
### MaskPSNRFine
For fine image, with mask

## SSIM / MaskSSIM
Same layout as PSNR, but use SSIM core calculation.
